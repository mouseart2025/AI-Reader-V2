# 样本小说说明

## 文件列表

| 文件名 | 类型 | 章节数 | 字数 | 用途 |
|--------|------|--------|------|------|
| 西游记-样本.txt | 修真/玄幻 | 25回(简化) | 约5千字 | 展示人物成长、门派体系、修炼等级 |
| 三国演义-样本.txt | 历史/战争 | 30回(简化) | 约6千字 | 展示复杂人物关系、势力分布、战争策略 |

## 设计目的

这些样本小说用于：

1. **零门槛体验**：用户安装后即可体验完整功能，无需准备自己的小说
2. **功能展示**：展示实体识别、关系提取、知识图谱构建等核心能力
3. **教学引导**：配合首次使用引导，帮助用户快速上手
4. **测试基准**：开发和测试时的标准数据集

## 样本内容特点

### 《西游记》样本
- **人物**：孙悟空、菩提祖师、唐僧、猪八戒、沙僧、白龙马等
- **地点**：花果山、水帘洞、灵台方寸山、斜月三星洞、天宫等
- **势力**：天庭、佛门、道门、花果山群妖
- **概念**：七十二变、筋斗云、长生不老、西天取经

### 《三国演义》样本
- **人物**：刘备、关羽、张飞、曹操、孙权、诸葛亮等
- **地点**：涿郡、桃园、虎牢关、洛阳、长安、许昌等
- **势力**：汉室、黄巾军、董卓军、袁绍军、曹操军、孙氏江东
- **概念**：三国鼎立、桃园结义、兵法谋略

## 版权说明

这些样本为**简化改编版**，用于产品功能演示：
- 原文为公版书（《西游记》- 明代吴承恩，《三国演义》- 明代罗贯中）
- 样本内容为结构化简化，保留核心人物、地点、情节
- 仅供AI Reader V2功能演示使用

## 使用方式

### 开发测试
```bash
# 启动后端
make dev

# 导入样本小说进行测试
curl -X POST http://localhost:8000/api/novels/upload \
  -F "file=@sample-novels/西游记-样本.txt"
```

### 首次启动
应用首次启动时自动加载样本小说，用户可立即体验：
1. 阅读功能
2. 实体高亮
3. 实体卡片
4. 关系图
5. 导出功能
